# Introduction
This folder contains a number of JSON schemas to define analytical services in the archaeometric laboratory. 
The main frameworks are taken from existing standards and have been modified by the author. 
The objective is to assemble metadata profiles of lab services. The users can copy and paste these JSON files onto an [online JSON-Form generator](https://bytes.kikirpa.be/) and save
the content as a JSON format to be furthered processed by software. 

### service_metadata/organsiation_metadata/method_metadata/person_metadata
These schemas are based on E-RHIS service models. 

### device_metadata
This schema follows the PIDINST standard

### software_metadata
This schema complies with Codemeta standard with some elements coming from E-RHIS software model

### reference_material_metadata
This schema is taken from the GeoREM database
